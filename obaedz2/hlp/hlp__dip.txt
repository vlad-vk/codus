~000 На экране определяются значения диапазонов каналов АСУТП.
~000 Модули УСО (Устройства Сопряжения с Объектом) имеют входной диапазон от 0 до 20 мА.
~000 Используемые приборы преобразования могут иметь выходной диапазон 0..5,0..20,4..20 мА.
~000 Для каждого канала нужно пересчитать диапазон шкалы отображения и сдвиг нуля.
~000 Для приборов с выходным диапазоном 4..20 мА пересчет будет следующим:
~000 Диапазон шкалы отображения= (шкала прибора) + (шкала прибора/4)
~000 Ноль шкалы отображения=   - (шкала прибора/4)
~000 Для приборов с выходным диапазоном 0..5 мА пересчет будет следующим:
~000 Диапазон шкалы отображения= (шкала прибора) * 4
~000 Ноль шкалы отображения= 0.
~000 В поле "Фильтр"  определяется максимальное значение на которое может изменяться
~000 текущее значение канала за один цикл программы контроллера.
~000 В поле "Среднее" определяют количество циклов за которые происходит  усреднение
~000 текущего значения измеряемого параметра.
~000 
~000 Для ввода значений, сначала, необходимо ввести пароль КИПиА.
~000 Для ввода пароля КИПиА нужно на клавиатуре нажать клавишу K (английскую) и, в
~000 появившемся окне, ввести пароль из шести цифр (пароль на экране не отображается).
~000 По окончании ввода пароля нужно нажать клавишу ENTER.

~000 Нажимая клавиши стрелочек на клавиатуре, надо установить двигающуюся рамочку на
~000 нужный параметр и нажать клавишу ENTER. При этом цифры в рамочке сдвигаются влево
~000 и появляется мигающий курсор для ввода значений.
~000 Он указывает какую цифру сейчас надо вводить.
~000 Введите нужное значение, нажимая клавиши на клавиатуре 0..9,-.
~000 По окончании ввода нажмите клавишу ENTER.   Введенное значение передается 
~000 на контроллер, программа контроллера его проверит и вернет для отображения.
~000 Если было набрано неправильное значение,  но  еще не была нажата клавиша ENTER,
~000 то для отмены набора можно нажать клавишу ESC.
~000 Если набранная цифра не вводится - значит набрано неправильное значение или не
~000 введен пароль КИП и А.
~000 После изменения значений нужно заблокировать пароль КИПиА, введя в поле пароля
~000 все нули (000000) и нажать клавишу ENTER.
