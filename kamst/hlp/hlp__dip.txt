~001^015 ЭКРАН: ~004ДИАПАЗОНЫ
~000
~000 На экране определяются значения диапазонов каналов АСУТП.
~000 Модули УСО (Устройства Сопряжения с Объектом) имеют входной диапазон от 0 до 20 мА.
~000 Используемые приборы преобразования могут иметь выходной диапазон 0..5,0..20,4..20 мА.
~000 Для каждого канала нужно пересчитать диапазон шкалы отображения и сдвиг нуля.
~000 Для приборов с выходным диапазоном 4..20 мА пересчет будет следующим:
~000 Диапазон шкалы отображения= (шкала прибора) + (шкала прибора/4)
~000 Ноль шкалы отображения=   - (шкала прибора/4)
~000 Для приборов с выходным диапазоном 0..5 мА пересчет будет следующим:
~000 Диапазон шкалы отображения= (шкала прибора) * 4
~000 Ноль шкалы отображения= 0.
~000 В поле "Фильтр"  определяется максимальное значение на которое может изменяться
~000 текущее значение канала за один цикл программы контроллера.
~000 В поле "Среднее" определяют количество циклов за которые происходит  усреднение
~000 текущего значения измеряемого параметра.
~000
