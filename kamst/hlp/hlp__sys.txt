~001^015 SCREEN: ~004SYSTEM
~000
~000 The SYSTEM screen displays the values the controller variables.
~000 This information is needed to set up the controller program.
~000 On this screen, the operator only looks at the trend (graph) of the network status.
~000 If a straight line is drawn on the trend, this indicates that the network
~000 is down or too slow.
~000
